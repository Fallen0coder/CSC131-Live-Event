# Live Event Web App

## Description
This project is a web application designed to help college students find local events and meet new people.

## Features
- User login and signup
- User profiles
- Browse nearby events
- View event details
- RSVP to events

## Tech Stack
- Frontend: Next.js
- Styling: Tailwind CSS
- Backend: (to be decided)
